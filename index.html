My Story so far:

I have been a young enthusiastic programmer since I was 12 years old when for the first time I approached to functional programming utilizing VB8, since then I amplified my information technology knowledge through various open source resources I found on the web.

When I arrived at the High school where I got my diploma I already knew basic knowledge about object oriented programmation and for school related studies I learned web developing and the basic concept about HTML, CSS and JavaScript for the Front-end part, furthermore I learned the basic concepts about relational databases and about SQL, utilizing MySql and phpMyAdmin using the local hosting software XAMPP.

Once I finished my studies I had an important work experience as a Full-stack PHP Developer at WeWine s.r.l. where I learned to work in team and to organize my own work in order to respect weekly deadlines.

Subsequently I continued to keep my self up to date regarding new web technologies finding a lot of satisfaction developing with FrameWorks such as Laravel for the back-end part with which I developed a management software, for the Front-End part I experimented with AngularJS, Bootstrap, Angular Material and JavaScript in the jQuery flavour.

Furthermore I attended a university level class where I learned Java's bases and I deepened my understanding and my knowledge about C and OOP concepts, even if for personal reasons I was unable to continue studies.
